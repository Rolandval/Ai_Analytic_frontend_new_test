## 🔄 Потік даних

### Перший візит (Cache Miss)

```
┌──────────┐
│          │
│  USER    │
│          │
└────┬─────┘
     │ 1. Відкриває сторінку
     ▼
┌─────────────────┐
│  Generation.tsx │
│  fetchData()    │
└────┬────────────┘
     │ 2. Перевірка кешу
     ▼
┌─────────────────┐
│  dataCache.ts   │
│  getCached...() │
└────┬────────────┘
     │ 3. Запит до IndexedDB
     ▼
┌─────────────────┐
│   IndexedDB     │
└────┬────────────┘
     │ 4. ❌ Немає даних
     ▼
┌─────────────────┐
│  Generation.tsx │
│  Показати лоадер│
└────┬────────────┘
     │ 5. Запит до API
     ▼
┌─────────────────┐
│   Backend API   │
└────┬────────────┘
     │ 6. Повернути дані
     ▼
┌─────────────────┐
│  Generation.tsx │
│  Показати дані  │
└────┬────────────┘
     │ 7. Зберегти в кеш
     ▼
┌─────────────────┐
│  dataCache.ts   │
│  cacheProducts()│
└────┬────────────┘
     │ 8. Зберегти в IndexedDB
     ▼
┌─────────────────┐
│   IndexedDB     │
│   ✅ Збережено  │
└─────────────────┘
```

### Наступні візити (Cache Hit)

```
┌──────────┐
│          │
│  USER    │
│          │
└────┬─────┘
     │ 1. Відкриває сторінку
     ▼
┌─────────────────┐
│  Generation.tsx │
│  fetchData()    │
└────┬────────────┘
     │ 2. Перевірка кешу
     ▼
┌─────────────────┐
│  dataCache.ts   │
│  getCached...() │
└────┬────────────┘
     │ 3. Запит до IndexedDB
     ▼
┌─────────────────┐
│   IndexedDB     │
└────┬────────────┘
     │ 4. ✅ Дані знайдено
     ▼
┌─────────────────┐
│  Generation.tsx │
│  ⚡ МИТТЄВО     │
│  показати дані  │
│  📦 Індикатор   │
└────┬────────────┘
     │
     │ 5. Користувач працює
     │
     │ (через 100ms)
     │
     │ 6. Фоновий запит
     ▼
┌─────────────────┐
│   Backend API   │
└────┬────────────┘
     │ 7. Свіжі дані
     ▼
┌─────────────────┐
│  Generation.tsx │
│  🔄 Оновити     │
│  ❌ Індикатор   │
└────┬────────────┘
     │ 8. Оновити кеш
     ▼
┌─────────────────┐
│   IndexedDB     │
│   ✅ Оновлено   │
└─────────────────┘
```

## ⏱️ Часова діаграма

```
ПЕРШИЙ ВІЗИТ (без кешу):
═══════════════════════════════════════════════════════════════

0ms    ┃ Користувач відкриває сторінку
       ┃
10ms   ┃ ⏳ Показати лоадер
       ┃
20ms   ┃ 🌐 Запит до API...
       ┃
       ┃ ⏳ Очікування...
       ┃
500ms  ┃ ✅ Отримано дані
       ┃
510ms  ┃ 📊 Показати таблицю
       ┃
520ms  ┃ 💾 Зберегти в кеш
       ┃
═══════════════════════════════════════════════════════════════
ЗАГАЛЬНИЙ ЧАС: ~520ms


НАСТУПНИЙ ВІЗИТ (з кешем):
═══════════════════════════════════════════════════════════════

0ms    ┃ Користувач відкриває сторінку
       ┃
10ms   ┃ 💾 Перевірка кешу
       ┃
20ms   ┃ ⚡ МИТТЄВО показати дані з кешу
       ┃ 📦 Показати індикатор
       ┃
       ┃ 👤 КОРИСТУВАЧ ВЖЕ ПРАЦЮЄ!
       ┃
120ms  ┃ 🔄 Фоновий запит до API...
       ┃
       ┃ (користувач не помічає)
       ┃
600ms  ┃ ✅ Отримано свіжі дані
       ┃
610ms  ┃ 🔄 Плавно оновити таблицю
       ┃ ❌ Приховати індикатор
       ┃
620ms  ┃ 💾 Оновити кеш
       ┃
═══════════════════════════════════════════════════════════════
ЧАС ДО ПЕРШОГО ПОКАЗУ: ~20ms (26x ШВИДШЕ!)
```

┌─────────────────────────────────────────────────────────────┐
│                      СТАНИ КЕШУ                              │
├─────────────────────────────────────────────────────────────┤
│                                                              │
│  1️⃣ EMPTY (Порожній)                                        │
│     ┌──────────────────────────────────────┐               │
│     │  • Перший візит                      │               │
│     │  • Кеш очищено                       │               │
│     │  • Дія: Завантажити з API            │               │
│     └──────────────────────────────────────┘               │
│                       │                                      │
│                       ▼                                      │
│  2️⃣ LOADING (Завантаження)                                  │
│     ┌──────────────────────────────────────┐               │
│     │  • Запит до API                      │               │
│     │  • Показати лоадер                   │               │
│     │  • Дія: Чекати відповіді             │               │
│     └──────────────────────────────────────┘               │
│                       │                                      │
│                       ▼                                      │
│  3️⃣ CACHED (Кешовано)                                       │
│     ┌──────────────────────────────────────┐               │
│     │  • Дані в IndexedDB                  │               │
│     │  • Вік < 24 години                   │               │
│     │  • Дія: Показати з кешу              │               │
│     └──────────────────────────────────────┘               │
│                       │                                      │
│                       ▼                                      │
│  4️⃣ REFRESHING (Оновлення)                                  │
│     ┌──────────────────────────────────────┐               │
│     │  • Показано дані з кешу              │               │
│     │  • Фоновий запит до API              │               │
│     │  • Індикатор "📦 Кеш"                │               │
│     └──────────────────────────────────────┘               │
│                       │                                      │
│                       ▼                                      │
│  5️⃣ FRESH (Свіжі дані)                                      │
│     ┌──────────────────────────────────────┐               │
│     │  • Отримано свіжі дані               │               │
│     │  • Оновлено кеш                      │               │
│     │  • Приховано індикатор               │               │
│     └──────────────────────────────────────┘               │
│                                                              │
└─────────────────────────────────────────────────────────────┘
