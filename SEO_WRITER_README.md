# SEO Writer & Auto Poster - Документація

## 📋 Огляд

**SEO Writer & Auto Poster** - це комплексна система управління контентом для соціальних мереж та блогу. Дозволяє генерувати теми, планувати публікації, редагувати статті та відстежувати аналітику.

## 🎯 Основні функції

### 1️⃣ Головна сторінка (Dashboard)
- **Статистика**: кількість запланованих, опублікованих та чернеток постів
- **Швидкий доступ**: кнопки для переходу до основних функцій
- **Останні теми та статті**: перегляд останніх створених елементів

**Маршрут**: `/seo-writer`

### 2️⃣ Генерація тем
- **Введення ключових слів**: форма для введення теми
- **Генерація ідей**: автоматична генерація тем на основі ключових слів
- **Управління темами**: затвердження, редагування, видалення тем
- **Платформи**: вибір платформ для публікації (Facebook, Instagram, LinkedIn, TikTok, Google Business, Blog)

**Маршрут**: `/seo-writer/topics`

### 3️⃣ Контент-план (Календар)
- **Календарний вигляд**: місячний календар з усіма постами
- **Фільтрація**: по статусу (запланировано, опубліковано, помилка)
- **Статуси постів**:
  - 🔵 Запланировано - пост готовий до публікації
  - 🟢 Опубліковано - пост успішно опубліковано
  - 🔴 Помилка - помилка при публікації

**Маршрут**: `/seo-writer/calendar`

### 4️⃣ Управління статтями
- **Таблиця статей**: перегляд всіх статей з фільтрацією
- **Пошук**: пошук по назві статті
- **Фільтрація**: по статусу (чернетка, на редагуванні, затверджено, опубліковано)
- **Дії**: перегляд, редагування, видалення статей
- **Версіонування**: відстеження версій статей

**Маршрут**: `/seo-writer/articles`

**Статуси статей**:
- 📝 Чернетка - початковий стан
- ✏️ На редагуванні - статтю редагують
- ✅ Затверджено - статтю затверджено
- 🚀 Опубліковано - статтю опубліковано

### 5️⃣ Аналітика
- **Метрики**: охоплення, лайки, коментарі, поділи, кліки
- **Графіки**: порівняння платформ, тренди залучення
- **Фільтрація**: по платформі та періоду
- **Розподіл**: діаграма розподілу охоплення по платформам

**Маршрут**: `/seo-writer/analytics`

### 6️⃣ Налаштування
- **Загальні**: мова, часовий пояс
- **Публікації**: автоматичний постинг, максимум постів на день
- **Сповіщення**: увімкнення сповіщень, щоденний дайджест
- **Безпека**: API ключ, зміна пароля

**Маршрут**: `/seo-writer/settings`

### API Структура

**seoWriterApi.ts** містить:

```typescript
// Типи
interface Topic { ... }
interface Article { ... }
interface ContentPlan { ... }
interface Analytics { ... }

// Mock дані
mockTopics: Topic[]
mockArticles: Article[]
mockContentPlan: ContentPlan[]
mockAnalytics: Analytics[]

// API функції
getTopics(): Promise<Topic[]>
getArticles(): Promise<Article[]>
getContentPlan(): Promise<ContentPlan[]>
getAnalytics(): Promise<Analytics[]>
generateTopics(keywords: string): Promise<Topic[]>
createArticle(...): Promise<Article>
updateArticle(...): Promise<Article>
publishArticle(...): Promise<Article>
```

### Типи даних

#### Topic
```typescript
{
  id: string
  title: string
  keywords: string[]
  category: string
  status: 'approved' | 'pending' | 'rejected'
  createdAt: Date
  platforms: Platform[]
}
```

#### Article
```typescript
{
  id: string
  topicId: string
  title: string
  content: string
  status: 'draft' | 'editing' | 'approved' | 'published'
  platforms: Platform[]
  scheduledDate?: Date
  publishedDate?: Date
  mediaUrls: string[]
  version: number
  createdAt: Date
  updatedAt: Date
}
```

#### ContentPlan
```typescript
{
  id: string
  date: Date
  articleId: string
  platforms: Platform[]
  status: 'scheduled' | 'published' | 'failed'
  scheduledTime: string
}
```

#### Analytics
```typescript
{
  postId: string
  platform: Platform
  reach: number
  likes: number
  comments: number
  shares: number
  clicks: number
  date: Date
}
```

#### Platform
```typescript
type Platform = 'facebook' | 'instagram' | 'linkedin' | 'tiktok' | 'google_business' | 'blog'
```

## 🔄 Робочий процес

### Типовий робочий процес

1. **Генерація тем**
   - Введіть ключові слова
   - Натисніть "Згенерувати ідеї"
   - Затвердіть теми

2. **Створення статей**
   - Виберіть тему
   - Напишіть або відредагуйте статтю
   - Додайте медіа-файли
   - Затвердіть статтю

3. **Планування публікацій**
   - Виберіть статтю
   - Встановіть дату та час публікації
   - Виберіть платформи
   - Запланіруйте публікацію

4. **Відстеження аналітики**
   - Перегляньте метрики по постам
   - Аналізуйте тренди
   - Оптимізуйте контент

## 📊 Мок-дані

Проект використовує мок-дані для демонстрації:

- **3 теми**: про сонячні панелі, акумулятори, інвертори
- **3 статті**: з різними статусами
- **3 контент-плани**: запланировані та опубліковані пости
- **2 записи аналітики**: з метриками по платформам

Для інтеграції з реальним backend:

1. **Замініть мок-дані** на реальні API виклики
2. **Оновіть API функції** в `seoWriterApi.ts`
3. **Додайте аутентифікацію** для захисту даних
4. **Налаштуйте обробку помилок** для реальних сценаріїв
