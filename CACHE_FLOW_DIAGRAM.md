## Легенда символів

```
⏳ - Лоадер (очікування)
⚡ - Миттєва дія
🌐 - Запит до API
💾 - Збереження в IndexedDB
🔄 - Оновлення даних
✅ - Успішна операція
❌ - Невдала операція / відсутність даних
📦 - Індикатор кешу
📝 - Логування
```
